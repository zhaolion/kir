=====================

一个私人定制的博客管理后台系统，希望能够做到简便部署，轻量运行，方便管理。

- front end: [hogo](https://github.com/gohugoio/hugo/)
- back end: kir

**技术栈**
- Golang 1.12+
- Vue

## 计划功能

- [] 文章展示
- [] 文章管理
- [] 分类管理
- [] 本地预览
- [] 一键部署
