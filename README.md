=====================


一个私人定制的博客系统，希望能够做到简便部署，轻量运行，方便管理

**技术栈**
- Golang 1.12+
- MySQL 5.7+
- Vue

## 计划功能

- [] 文章展示
- [] 文章管理
- [] 分类管理
- [] 系统设置
- [] 一键部署
- [] 管理后台
- [] 数据库升级
